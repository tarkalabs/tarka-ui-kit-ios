# Tarka UI Kit iOS
Tarka UI Kit is a reusable component library for building iOS and iPadOS apps, based on a design system using Atomic Design principles

# List of components
- TUIAvatar
- TUIBadge
- TUICircularProgressView
- TUIIconButton
- TUIButtonBlock
- TUIDraggableCard
- TUIMenuItemView
- TUIOverlayFooter
- TUISelectionRow
- TUINavigationRow
- TUITextRow
- TUISnackBar
- TUIStatusIndicator
- TUITabBar
